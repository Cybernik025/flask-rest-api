# Flask REST API

A RESTful API built with Flask to manage tasks. Designed to demonstrate clean architecture, Docker integration, and debugging setup.

## Features
- User registration and login (JWT-based authentication)
- CRUD operations for tasks
- Input validation and error handling
- Modular, scalable Flask project structure
- Dockerized for easy deployment and development
- Debugging configured for local development

## Tech Stack
- Python 3.11
- Flask
- SQLAlchemy / SQLite
- Pytest for testing
- Docker & Docker Compose
